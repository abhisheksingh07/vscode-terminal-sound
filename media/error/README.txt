DROP YOUR CUSTOM ERROR SOUNDS HERE
====================================
Place any .mp3 or .wav files in this folder.
They will automatically appear in the sound selector.

To select a sound:
  Open Command Palette (Ctrl+Shift+P) → "PlaySound: Select Error Sound"

Built-in sounds:
  error-1.mp3   - Original buzzer
  error-2.mp3   - Short beep (440Hz)
  error-3.mp3   - Low tone (300Hz)
  error-sound.mp3 - Classic beep
